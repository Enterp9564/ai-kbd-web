명령어 가이드 (RAW 명령 탭)

웹앱의 원문 명령 입력창은 문자열을 그대로 Pi GATT 서버로 보냅니다.

기본 명령

  • PING: 연결 왕복 테스트 (PONG 응답)
  • STATUS: HID 상태 확인 (예: OK:LIVE:/dev/hidg0:lang=en)

입력 명령

  • TEXT:Hello 문자열 입력
  • TEXT:저는 KDS입니다. 한글/영문 혼용 입력
  • KEY:ENTER 단일 키 입력
  • KEY:BACKSPACE 백스페이스 입력
  • KEY:CTRL+B 조합키 입력

지원 KEY

  • ENTER, ESC, BACKSPACE, TAB, SPACE
  • UP, DOWN, LEFT, RIGHT
  • F1~F12, 0~9, A~Z
  • HANGEUL, HAN/ENG
  • 조합키: CTRL+B, CTRL+I, CTRL+U, ALT+TAB

TEXT 특수문자 처리

  • ASCII/한글(2벌식 변환 가능 범위)은 그대로 입력합니다.
  • 일부 특수문자는 자동 치환합니다. 예: · -> /, 「」 -> [], ※ -> *
  • 치환 규칙이 없는 문자는 ?로 대체하여 전송을 중단하지 않습니다.

오류 응답

  • ERR:UNKNOWN_CMD: 알 수 없는 명령 형식
  • ERR:PermissionError: /dev/hidg0 접근 권한 문제
  • ERR:ValueError: 잘못된 키 이름/값

대용량 TEXT 전송

  • TEXT 명령이 512바이트를 넘으면 자동 분할 전송됩니다.
  • 전송 전 확인 창이 표시됩니다.
